# 🌊 Seashell Stories – Luxury Café Website

A modern **luxury café website UI** built using React + Vite + Tailwind CSS.

## ✨ Features
- Luxury cinematic hero section
- Smooth scroll navigation
- 3D hover menu cards
- Glassmorphism UI design
- Animated ocean-themed background
- Working reservation form with popup
- Fully responsive design

## 🛠 Tech Stack
- React.js
- Vite
- Tailwind CSS
- Framer Motion
